## 📋 项目概述

CCS-Tender是一款专为China Comservice开发的专业招标文件法务与商务条款智能分析桌面应用。该系统采用先进的AI技术,结合多API自动切换机制,为企业提供高效、准确、专业的招标文件分析服务。

### 核心价值

- **智能化**: 采用GPT-4.1-mini和Gemini 2.5-flash双引擎,确保分析质量
- **专业性**: 针对不同国家法律体系进行精准分析
- **可靠性**: 多API自动切换,保证服务高可用性
- **易用性**: 高端专业的界面设计,操作简单直观
- **安全性**: 本地处理,不上传敏感数据

## 🚀 部署与使用

### 开发环境

#### 安装依赖
```bash
cd frontend
pnpm install
```

#### 启动开发服务器
```bash
pnpm run electron:dev
```

#### 启动Web开发服务器(仅前端)
```bash
pnpm run dev
```

### 生产环境

#### 构建前端
```bash
pnpm run build
```

#### 打包Windows应用
```bash
pnpm run package:win
```

#### 输出文件
- `dist-electron/CCS-Tender-1.0.0-Setup.exe` - 安装程序(~150MB)
- `dist-electron/CCS-Tender-1.0.0-Portable.exe` - 便携版(~150MB)

### API配置

#### 设置环境变量
```powershell
# Windows PowerShell
[System.Environment]::SetEnvironmentVariable("OPENAI_API_KEY", "sk-your-key", "User")
[System.Environment]::SetEnvironmentVariable("GEMINI_API_KEY", "your-key", "User")
```
