## 系统简介

CCS-Tender是China Comservice专业开发的招标文件智能分析系统,旨在为企业提供高效、准确的招标文件法务与商务条款分析服务。

### 核心优势

- **智能分析**: 采用先进的AI技术,自动识别法务风险和商务条款问题
- **多语言支持**: 支持中英文界面切换,自动识别文件语言
- **专业报告**: 生成符合国际标准的PDF分析报告
- **高可用性**: 多API自动切换,确保服务稳定性
- **隐私保护**: 本地处理,不上传敏感数据

## 安装指南

### 系统要求

- **操作系统**: Windows 11 (64位)
- **处理器**: Intel Core i3或更高
- **内存**: 4GB RAM (推荐8GB)
- **硬盘空间**: 500MB可用空间
- **网络**: 稳定的互联网连接

### 安装步骤

#### 方式一:安装版

1. 双击 `CCS-Tender-1.0.0-Setup.exe`
2. 选择安装语言(中文/English)
3. 选择安装路径
4. 点击"安装"按钮
5. 等待安装完成
6. 点击"完成"启动应用

#### 方式二:便携版

1. 双击 `CCS-Tender-1.0.0-Portable.exe`
2. 应用直接启动,无需安装

### API密钥配置

使用前需要配置API密钥:

1. 打开Windows PowerShell(管理员模式)
2. 输入以下命令:

```powershell
[System.Environment]::SetEnvironmentVariable("OPENAI_API_KEY", "your-api-key", "User")
[System.Environment]::SetEnvironmentVariable("GEMINI_API_KEY", "your-api-key", "User")
```

3. 重启应用使配置生效

## 快速开始

### 第一次使用

1. **启动应用**
   - 双击桌面快捷方式"CCS-Tender"
   - 或从开始菜单启动

2. **选择语言**
   - 点击右上角语言切换按钮
   - 选择中文或English

3. **上传文件**
   - 点击上传区域
   - 选择招标文件(PDF/Word/Excel)
   - 或直接拖拽文件到上传区域

4. **等待分析**
   - 系统自动显示上传进度
   - 上传完成后自动开始分析
   - 分析进度实时显示

5. **下载报告**
   - 收到完成提醒通知
   - 点击"下载PDF报告"按钮
   - 选择保存位置

### 典型工作流程

```
上传文件 → 文件解析 → AI分析 → 生成报告 → 下载报告
   ↓          ↓          ↓          ↓          ↓
 进度显示   语言识别   风险识别   图表生成   提醒通知
```

## 功能详解

### 1. 文件上传

#### 支持格式
- PDF文档(.pdf)
- Word文档(.doc, .docx)
- Excel表格(.xls, .xlsx)

#### 上传方式
- **点击上传**: 点击上传区域,选择文件
- **拖拽上传**: 直接拖拽文件到上传区域

#### 文件限制
- 单个文件最大50MB
- 建议文件内容清晰,格式规范

### 2. 智能分析

#### 分析内容

**法务风险分析**
- 合同条款合规性检查
- 法律风险点识别
- 违约责任条款分析
- 争议解决机制评估

**商务条款分析**
- 付款条件评估
- 交付时间合理性
- 违约金条款分析
- 保修保证条款审查
- 价格成本结构分析

#### 分析流程

1. **国家法律体系识别** (20%)
   - 识别合同所属国家/地区
   - 确定适用法律体系
   - 识别监管框架

2. **法务风险评估** (40%)
   - 识别高/中/低风险点
   - 提供风险描述
   - 给出专业建议

3. **商务条款审查** (60%)
   - 提取关键商务条款
   - 分析条款合理性
   - 标注警示信息

4. **综合评估** (80%)
   - 生成执行摘要
   - 计算合规性评分(0-100)
   - 提供改进建议

5. **数据可视化** (100%)
   - 生成风险分布图表
   - 创建合规性雷达图
   - 统计分析结果

### 3. 报告生成

#### 报告结构

**第1页:封面**
- 企业LOGO和名称
- 报告标题
- 文档信息(文件名、类型、字数等)
- 生成日期和时间

**第2页:执行摘要**
- 分析概要
- 合规性评分
- 评分可视化条

**第3页:法务风险分析**
- 风险统计(高/中/低)
- 详细风险列表
- 风险描述和建议

**第4页:商务条款分析**
- 关键条款列表
- 条款值和分析
- 警示标注

**第5页:建议与行动计划**
- Top 5建议
- 具体行动步骤

**第6页:数据可视化**
- 风险分布图
- 类别统计图
- 合规性雷达图

#### 报告特点
- 专业排版,图文并茂
- 中文字体嵌入,无乱码
- 符合国际标准
- 可打印,可存档

### 4. 多语言支持

#### 界面语言
- 中文(简体)
- English(英文)

#### 切换方法
- 点击右上角地球图标
- 选择目标语言
- 界面即时切换

#### 文件语言识别
- 自动识别上传文件语言
- 支持中英文文档
- 自动转换为通用语言分析

### 5. 提醒功能

#### 提醒类型

**上传完成提醒**
- 文件上传100%时触发
- 播放提示音
- 显示系统通知

**分析完成提醒**
- 分析进度100%时触发
- 播放提示音
- 显示系统通知
- 弹出下载按钮

#### 通知方式
- 系统托盘通知
- 音效提示
- 应用内弹窗

## 常见问题

### Q1: 应用无法启动?

**解决方案**:
1. 检查系统是否为Windows 11 (64位)
2. 确认已安装所有系统更新
3. 以管理员身份运行
4. 重新安装应用

### Q2: 上传文件失败?

**解决方案**:
1. 检查文件格式是否支持(PDF/Word/Excel)
2. 确认文件大小不超过50MB
3. 检查文件是否损坏
4. 尝试转换文件格式后重新上传

### Q3: 分析过程中断?

**解决方案**:
1. 检查网络连接是否稳定
2. 确认API密钥配置正确
3. 查看系统日志获取详细错误信息
4. 重新上传文件重试

### Q4: PDF报告出现乱码?

**解决方案**:
1. 确认使用最新版本应用
2. 检查PDF阅读器是否支持中文
3. 尝试使用Adobe Acrobat Reader打开
4. 联系技术支持

### Q5: API密钥如何获取?

**解决方案**:
1. **OpenAI API**:
   - 访问 https://platform.openai.com
   - 注册账号并创建API密钥

2. **Gemini API**:
   - 访问 https://ai.google.dev
   - 注册Google Cloud账号
   - 创建API密钥

### Q6: 如何更新应用?

**解决方案**:
1. 下载最新版本安装包
2. 运行安装程序
3. 选择"覆盖安装"
4. 保留原有配置和数据
