## 部署说明

### 开发环境部署

1. **安装依赖**
```bash
cd frontend
pnpm install
```

2. **配置API密钥**
```bash
# Windows PowerShell
$env:OPENAI_API_KEY="your-api-key"
$env:GEMINI_API_KEY="your-api-key"
```

3. **启动开发服务器**
```bash
pnpm run electron:dev
```

### 生产环境构建

1. **构建前端**
```bash
pnpm run build
```

2. **编译Electron**
```bash
tsc electron/main.ts --outDir electron
```

3. **打包Windows应用**
```bash
pnpm run electron:build
```

4. **输出文件**
- `dist-electron/CCS-Tender-1.0.0-Setup.exe` - 安装程序
- `dist-electron/CCS-Tender-1.0.0-Portable.exe` - 便携版

## 已知问题与限制

### 限制

1. **文件大小**: 单个文件最大50MB
2. **网络依赖**: 分析功能需要稳定的网络连接
3. **API限制**: 免费API有调用频率限制
4. **语言支持**: 当前仅支持中英文

### 待优化项

1. **离线模式**: 未来版本考虑支持离线分析
2. **批量处理**: 未来版本支持批量文件分析
